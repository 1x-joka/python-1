# PYTHON: BÁSICO AO INTERMEDIÁRIO

> Aulas do Curso em Vídeo com total foco em Python

---

## Etapas do Aprendizado

- **Mundo 1** Fundamentos: Tipos Primitivos, Operadores Aritméticos, Módulos, Manipulação de Texto, Condições, Utilização de Cores, entre outros.
- **Mundo 2** Estruturas de Controle: Condições Aninhadas, Estrutura de Repetição "FOR", Estrutura de Repetição "WHILE", Interrupção de Repetições "WHILE", entre outros.
- **Mundo 3** Estruturas Compostas: Listas, Dicionários, Funções, Módulos e Pacotes, Tratamento de Erros e Exceções, entre outros.
